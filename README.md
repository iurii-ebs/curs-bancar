### Currency parser
### banks short names - MAIB, MICB, Mobias, BNM

### endpoints:

### parse data from all banks
```
banks/get/all/
```

### parse data for a specific date
```
banks/get/all/?date=2020-07-22 
```

### Register new user 
```
api/user/register/        
```
### Get Authorization Bearer Token
```
aip/user/token/           
```
